# Nutrition Tracking App

## Overview
A nutrition and body composition tracking application inspired by MacroFactor and MacroPhase. Users can track food intake, weight trends, and progress toward fitness goals.

## Tech Stack
- **Frontend**: Next.js (React / TypeScript), Tailwind CSS, Recharts, React Query
- **Backend**: Node.js / Express, TypeScript, SQLite (dev) / PostgreSQL (prod)
- **DevOps**: ESLint, Prettier, Jest, Git
- **Deployment**: Vercel (frontend), Render / Railway (backend)

## Design Principles
- Dark mode first
- Glassmorphism only where tasteful
- Gorgeous typography
- Excellent spacing
- Smooth animations
